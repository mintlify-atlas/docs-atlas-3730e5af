Stremio supports keyboard shortcuts for navigating the interface and controlling playback. Shortcuts are available on desktop browsers and in the desktop app. They are not available on mobile.

<Info>
  Keyboard shortcuts are disabled when the cursor is inside a text input field.
</Info>

## Viewing shortcuts in the app

You can open the shortcuts reference at any time by pressing `Ctrl` + `/`. The shortcut list is also available in **Settings → Shortcuts**.

## General navigation

These shortcuts work anywhere in the Stremio interface.

| Key | Action |
|-----|--------|
| `0` | Go to search |
| `1` | Go to the Board (home) |
| `2` | Go to Discover |
| `3` | Go to Library |
| `4` | Go to Calendar |
| `5` | Go to Addons |
| `6` | Go to Settings |
| `F` | Toggle fullscreen |
| `Escape` | Go back / exit current screen |
| `Ctrl` + `/` | Show keyboard shortcuts reference |

<Note>
  The number keys `1`–`6` navigate directly to the corresponding sidebar tab. `0` always goes to search regardless of position.
</Note>

## Player controls

These shortcuts are active while a video is open in the player.

### Playback

| Key | Action |
|-----|--------|
| `Space` | Play / Pause |
| `Shift` + `N` | Play next video |

### Seeking

| Key | Action |
|-----|--------|
| `→` | Seek forward (short duration) |
| `Shift` + `→` | Seek forward (long duration) |
| `←` | Seek backward (short duration) |
| `Shift` + `←` | Seek backward (long duration) |

The seek durations are configurable in **Settings → Player → Controls**. See [seek duration](/configuration/player-settings#seek-duration) for available values.

### Volume

| Key | Action |
|-----|--------|
| `↑` | Volume up |
| `↓` | Volume down |
| `M` | Toggle mute |

### Subtitles

| Key | Action |
|-----|--------|
| `C` | Toggle subtitles on / off |
| `S` | Open subtitles menu |
| `=` | Increase subtitle size |
| `-` | Decrease subtitle size |
| `G` | Increase subtitle delay |
| `H` | Decrease subtitle delay |

### Menus

| Key | Action |
|-----|--------|
| `A` | Open audio track menu |
| `I` | Open info / episodes panel |
| `R` | Open playback speed menu |
| `D` | Open statistics panel |

## Platform notes

<AccordionGroup>
  <Accordion title="macOS">
    All shortcuts listed above work on macOS. There are no macOS-specific modifier substitutions — `Ctrl` is used as written, not `Cmd`.
  </Accordion>
  <Accordion title="Windows and Linux">
    All shortcuts work as described. The `Ctrl` key is the standard modifier.
  </Accordion>
  <Accordion title="Mobile (iOS, Android, visionOS)">
    Keyboard shortcuts are not available on mobile platforms. The Shortcuts section does not appear in Settings on mobile devices.
  </Accordion>
</AccordionGroup>
